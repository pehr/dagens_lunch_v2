# Dagens Lunch

This project is in two parts. A backend, run on AWS and set up with CDK/Cloudfront. and a Frontend web application, using the APIs provided by the backend.

## Frontend
if needed. To install Tailwind 3
```
npm install -D tailwindcss@3      
```
To compile tailwind output:

```
npx tailwindcss -i styles.css -o dist/output.css
```
